# sistema-estoque-python
# Sistema de Controle de Estoque em Python

Projeto desenvolvido para praticar lógica de programação e estruturação de sistemas utilizando Python.

## Funcionalidades

- Cadastro de produtos
- Remoção de produtos
- Atualização de preços
- Atualização de estoque
- Visualização do estoque
- Tratamento de erros com try/except
- Menu interativo no terminal

## Tecnologias utilizadas

- Python
- Dicionários
- Funções
- Estruturas de repetição
- Try/Except

## Objetivo

Praticar desenvolvimento em Python com foco em evolução para a área de Análise de Dados.
